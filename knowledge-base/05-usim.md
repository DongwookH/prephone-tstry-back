# 05. 유심 호환 (USIM)

## 사용 가능한 유심 (정확)

| 유심 | 가격 | 구매처 |
|------|------|--------|
| **KT 바로유심** (K망) | 8,800원 | 이마트24, CU, GS25 편의점 |
| **LG U+ 모두의유심원칩** (L망) | 8,800원 | 이마트24, 스토리웨이, 배민 B마트, 지하철 자판기 |

## 사용 불가능한 유심
- 이외 타 알뜰폰 유심: **사용 불가**
- SKT 통합유심: 호환 X
- 기존 통신사 사용 유심: 호환 X
