# 07. 글 작성 규칙 (Content Rules)

## 톤앤매너
- **친근한 존댓말** (~해요, ~끝나요, ~기억해 주세요)
- 짧은 문장 (12~18자 평균)
- 이모지 활용: 📌 ✅ 💬 📱 🔍 ⬅️ ⚠️ 🔑

## 반드시 반복 강조할 핵심 메시지 (3~5회)
1. **5분 비대면 셀프개통**
2. **신용조회 없음** (신용불량자도 OK)
3. **약정·위약금 없음** (일반 요금제)
4. **승인 대기 없음** (홈페이지 개통 시)
5. **앤텔레콤 안심개통 케어통신** (브랜드)

## 금지 표현 (절대 사용 X)
- "100% 보장", "100% 가능" — 과장 광고 금지
- "무조건 가능" — 케이스별 예외 있음
- "다른 업체는 안 됨" — 비방 금지
- 경쟁사 명칭 직접 비방
- 외부 링크 인용 (자제)

## 권장 표현
- "안심개통"
- "비대면 셀프"
- "케어통신"
- "5분이면 끝"
- "꼭 기억해 주세요"

## 가격·숫자 표기
- 가격: `12,100원` / `33,000원` (콤마 포함)
- 시간: `5분`, `5~10분` (한국식)
- 전화: `010-2311-6543` 또는 `010 2311 6543`
- 고객센터: `114 (무료)` / `1899-7700 (유료)`

## ⚠️ 사실 검증 룰
- **가격, 정책, FAQ 답변은 무조건 KB(02-plans.md, 04-faq.md) 그대로** — 추측·창작 금지
- 모르는 정보는 "자세한 내용은 [공식 사이트](https://ntelecomsafe.com) 또는 [카톡 문의](https://pf.kakao.com/_Sxmxixon/chat)에서 확인" 식으로 우회
- 외부 사이트 인용 자제 (KAIT, 방통위 등 매우 신중하게)
