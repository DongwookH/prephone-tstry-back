# 03. 개통 절차 (Process)

## 비대면 셀프개통 단계 (정확한 순서)

1. **홈페이지 접속** (https://ntelecomsafe.com)
2. **안면인식** 본인 확인
3. **간편인증** (PASS / 카카오 / 토스 / KB / SOL 중 택1)
4. **유심번호 입력**
   - K망: KT 바로유심
   - L망: LG U+ 모두의유심원칩
5. **신분증 정보 기재** (촬영 후 정보 입력)
6. **요금제 선택**
7. **개통 완료 및 충전** (평균 5~10분 내 사용 시작)

## 필요한 본인인증 수단
| 인증 수단 | 가능 여부 |
|----------|---------|
| PASS | ✅ |
| 카카오인증 | ✅ |
| 토스인증 | ✅ |
| KB(국민) | ✅ |
| SOL(신한) | ✅ |

## 결제 방법
| 결제 수단 | 가능 여부 |
|----------|---------|
| 카드 | ✅ |
| 계좌이체 | ✅ |
| 카카오페이 | ✅ |
| 무통장 입금 | ✅ |

## 셀프개통 가능 시간
- **번호이동**: 10:00~19:50 (일요일·설·추석 당일 불가)
- **신규가입**: 08:00~21:50

## 셀프개통이 안 되는 경우 (반드시 매장 방문)
1. 외국인 (여권 소지자)
2. 미성년자
3. 법인
4. 타사 번호이동·신규가입·명의변경 후 3개월 이내 회선
5. 30일 이내 앤텔레콤 번호를 해지한 경우

## 번호이동 인증 제한
- 하루 **최대 7회**까지 시도 가능
- 7회 초과 시 당일 번호이동 불가 → 다음 날 재시도
