# 01. 제공 서비스 (Services)

## 제공하는 서비스
- 선불폰 비대면 셀프개통
- 전국 선불폰 매장 방문 개통 예약 안내 (80여 개 매장)

## 처리 가능한 상황 (사실 — 추측 금지)

| 케이스 | 가능 여부 | 비고 |
|--------|----------|------|
| 신용불량자 | ✅ 가능 | 신용조회 없음 |
| 통신비 미납자 | ✅ 가능 | 타사 미납 OK, 자사 미납은 불가 |
| 수신정지 폰 | ✅ 가능 | 즉시 해결 |
| 발신정지 폰 | ✅ 가능 | 즉시 해결 |
| 직권해지 폰 | ✅ 가능 | |
| 외국인 (여권) | ⚠️ 매장 방문 필수 | 비대면 불가 |
| 외국인 (외국인등록증/거소증) | ✅ 간편인증서 있으면 비대면 가능 | |
| 미성년자 | ⚠️ 보호자 동행으로 가능 | 비대면 불가 |
| 법인 | ❌ 선불 불가 | 후불 알뜰폰 최대 4회선까지 가능 |
| 명의도용/대포폰 의심자 | ❌ 개통 불가 | |

## 평균 처리 시간
- **비대면 셀프개통**: 5~10분
- **승인 대기**: 홈페이지 개통 시 승인 대기 **없음**
- **충전 반영**: 평균 5~10분
