# 06. 케이스별 처리 (Cases)

## 외국인
- **가능한 비자**: 모두 가능 (F-2, F-4, F-5, E-7, D-2, 등)
- **불가능한 케이스**: 불법체류자
- **필요 서류**: 여권 / 외국인등록증 / 외국인거소증 중 1택
- **개통 방법**:
  - **여권 소지자**: 매장 방문 필수 (비대면 불가)
  - **외국인등록증/거소증 소지자**: 지정 간편인증서 있으면 비대면 가능, 없으면 매장 방문
- **회선 한도**: 1회선

## 신용불량자
- **신용조회 없음** → 무조건 가능
- 타사 미납 있어도 OK
- ⚠️ 단, **앤텔레콤 자사 미납이 있는 경우 개통 불가**

## 미성년자
- **만 14세 이상**: 본인 명의 개통 가능 (보호자 동행 필수)
- **만 14세 미만**: 법정대리인을 통한 개통
- **회선 한도**: 선불·후불 각 1회선
- **필요 서류**:
  - 법정대리인 직접 내방: 법정대리인 신분증, 가족관계확인서(3개월 이내), 미성년자 신분증, 고객본인확인서
  - 대리인 내방: 인감증명서, 인감도장 날인 위임장, 대리인 신분증 추가

## 법인
- **선불폰**: 개통 불가
- **후불 알뜰폰**: 최대 4회선까지 가능
- **설립 6개월 미만 또는 유한회사**: 1회선만 가능

## 통신비 미납자
- **타 통신사 미납**: 개통 가능 (선불은 신용조회 X)
- **앤텔레콤 자사 미납**: 개통 불가

## 정지·해지 상태
- **수신정지 폰**: 선불폰으로 즉시 해결 가능
- **발신정지 폰**: 선불폰으로 즉시 해결 가능
- **직권해지 폰**: 선불폰으로 개통 가능
- **30일 이내 앤텔레콤 해지 이력**: 셀프개통 불가, 매장 방문 필요

## 명의도용/대포폰 의심자
- 개통 **불가** (법적 보호)

## 회선 제한 총정리

| 대상 | 최대 회선 | 비대면 가능 |
|------|---------|-----------|
| 내국인 (신용 양호) | 3회선 | ✅ |
| 미성년자 | 선불·후불 각 1회선 | ❌ (매장 방문) |
| 외국인 (등록증+간편인증) | 1회선 | ✅ |
| 외국인 (여권) | 1회선 | ❌ (매장 방문) |
| 법인 | 후불 4회선 | ❌ (선불 불가) |
| 법인 (6개월 미만/유한회사) | 후불 1회선 | ❌ |
